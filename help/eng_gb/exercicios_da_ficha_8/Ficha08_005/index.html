Height of a Person

Problema Fluxograma Algorithmi
Build an algorithm that asks the user for the height of a person in meters. The height of a person varies between 20 cm and 3 meters.

 
    begin MainProgram
        define real height = 0.0
        do
            write "Height : "
            read height
        while height < 0.2 || height > 3.0
        write "Height = " + height
    end MainProgram
