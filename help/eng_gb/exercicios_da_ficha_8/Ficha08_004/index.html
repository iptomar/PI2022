Positive number

Problema Fluxograma Algorithmi
Build an algorithm that asks the user for an integer greater than zero.

 
    begin MainProgram
        define integer n = 0
        do
            write "n : "
            read n
        while n <= 0
        write "Value of n = " + n
    end MainProgram
