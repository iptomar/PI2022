Write
Writes the value of one or more expressions to the console. Expressions can be made up of simple variables, in which case it writes its content, or a complex expression, in which case it writes the result after being converted to text

Syntax 

write <expression>[, <expression> …]


<TYPE> – Data type (integer | real | logical | text)

<expression>– expression computacional

Example

Write a phrase "Hello World" 
 write "Hello World"
Write a text variable ( type omission) 
 define text txt = "Algorithmi"

 Write txt
