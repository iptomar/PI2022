3.2.4 Iterating a set of values

Lets you iterate over a set of values within the range [ex1, ex2 [ where values are separated by ex3. The starting iterator limit is included and the ending limit is not. The instruction allows you to define na iteration variable or use and existing numeric variable.

Algorithmi
BEGIN MainProgram
ITERATE INTEGER i FROM 0 TO 10 STEP 1
WRITE i SUM " "
END ITERATE
END MainProgram
Fluxogram Pseudocode

(Type) – numeric variable type( INTEGER | REAL) - Opcional
(var) - iteration variable name
(ex1) – computational expression to determine the initial value
(ex2) – computational expression to determine the final value
(ex3) – computational expression to determine the separation
between the range values

3.2.2Repetition with initial control – While …doing

Allows you to repeat a set of intructions based on a condition that is evaluated at the beginning of the repetition.

Algorithmi
BEGIN MainProgram
DEFINE INTEGER i SET 0
WHILE i less than 10
WRITE i SUM " "
EXECUTE i SET i SUB 1
END WHILE
END MainProgram
