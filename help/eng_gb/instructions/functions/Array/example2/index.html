Array

Calculation and printing of a two-dimensional array
Flowchart Pseudocode 
            begin MainProgram
                define integer array[8][4]
                for integer i from 0 to elementsOf(array) step 1
                    for integer j from 0 to length(array[i]) step 1
                        execute array[i][j]=i*10+j
                        write "\t"+array[i][j]
                    end for
                write "\n"
                end for
            end MainProgram
