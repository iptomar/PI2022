Functions with Return
Functions with empty return, usually called procedures, serve to execute a set of instructions. These functions can only be invoked from an EXECUTE command

Syntax 

Function
function <Type> <name> ([= <parameter>,...,<parameter>]) 
 <instructions> 
end <name>


<TYPE> – Tipo de dados (intenger| float | string | void)

 o empty means it does not calculate anything <name> – Variable name subject to the following rules
 o Cannot be a reserved word or other function
 o Must start with a letter (a-z ; A-Z)
 o May contain digits (0-9)
 o Must be significant of what stores

[<parameters>] - Function parameters(Optional)
 o <Type> Parameter data type
 o <Var>Parameter name

Examples

EXAMPLE 1 - Function void sayHello() 
begin MainProgram
    execute sayHello()
end MainProgram

function void sayHello()
    write "Hello World"
end sayHello
EXAMPLE 2 - Function void function01() 
begin MainProgram
    execute function01()
end MainProgram

function void function01()
    write "Hello World of functions"
end function01
EXAMPLE 3 - Function integer summation(integer n) 
begin MainProgram
    define integer stop = 5
    write "\n" + summation(stop)
end MainProgram

function integer summation(integer n)
    define integer sum = 0
    for integer i from 0 to n  step 1
        write i + " "
        execute sum = sum + i
    end for
    return sum
end summation
