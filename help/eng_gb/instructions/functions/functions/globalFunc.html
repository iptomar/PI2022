Global memory
Each function has its memory that can be used by its instructions. However, algorithmi allows defining a global memory that is common to all functions and can be used by them.

Syntax 

        Syntax Text
memory Global
    define <type> <namevar> = 1.0
end Global

<type> – Parameter data type

<namevar> – Variable name

Examples:

EXAMPLE 1 - Global memory usage 
memory Global
    define real ver = 1.0
end Global

begin MainProgram
    define string prog = " algorithmi"
    for integer i from 1 to 10 step 1
        write sayHello(i + prog)
end MainProgram

function string sayHello(string name)
    define string msg = "Olá " + name
    return msg + ver + "\n"
end sayHello
