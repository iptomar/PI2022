Function with parameters
The parameters of a function work like local variables of the function and serve to make its usefulness flexible.

Sintaxe 

Function
Function <Type> <name> ([= <parameter>,...,<parameter>]) 
 <instructions> 
end <name>


<TYPE> – type of data (integer | float | string | void)

 o empty means it does not calculate anything <name> – Variable name to the following rules
 o Cannot be a reserved word or other function
 o Must start with a letter (a-z ; A-Z)
 o May contain digits (0-9)
 o Must be significant of what stores

[<parameter>] - Function parameters (Optional)
 o <Type> Parameter data type
 o <Var>Name of parameter
