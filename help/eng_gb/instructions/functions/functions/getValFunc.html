Get a value from a function

Functions can perform calculations that are returned to the location where the function was invoked.
The RETURN statement allows returning a value from the function.
Whenever this instruction is executed the function ends.

Syntax 

        Syntax Text

return <expression>

<expression> – Computational expression compatible with the function's return data type
 o If the function returns Empty , the expression must be empty.

Examples:

EXAMPLE 1 - Say hello with parameters
    return "Hello " + name
