Functions
Functions do not function as subs that can be invoked within instructions.

Syntax 

        Syntax Text
function <TYPE> <name> ([<param>,..., <param>
   <instructions> …
end name_Function


<TYPE> – type of calculation performed by the function

<Name> – Role name subject to the following rules
 o Cannot be a reserved word or other function
 o Must start with a letter (a-z ; A-Z)
 o May contain digits (0-9)
 o Must be significant of what stores

[<Param>] - Function Parameters (Optional)
 o type –> Parameter data type
 o Variables -> Parameter name

Examples:

EXAMPLE 1 define function and use it 
begin MainProgram
    execute function01()
end MainProgram

function void function01()
    write "Hello world of functions"
end function01
