Execute
It serves to perform calculations on computational expressions and store their result in a variable defined in memory. It also serves to execute procedures, that is, functions that do not calculate values. This one with the multi-statement allows you to do multiple calculations in one form.

Syntax 

run <function> | <var> = <expression>[, <function> | <var> = <expression> …] ]


<TYPE> – • – Data type (INTEGER | REAL | LOGIC | TEXT)

• <var> – variable defined in memory
• <expression> – computational expression
• <function> – user defined function
The symbol “|” represents “or”, that means it can be <function> ou <var> = <expression>

Exemplos

EXEMPLO 1
Read 2 variables (n1,n2), set one of these variables as the sum of both using the execute statement and write that value 
    read integer n1 "1st number : "
    read integer n2 "2nd number : "
    run n1 = n1 + n2
    write "Sum = " + n1
EXAMPLE 2
Define two integer variables, var1 with the value 10 and i with the value of 0, then create a cycle while...does where you will use the execute instruction to change the value of the variable i and finally use an instruction write outside the cycle to check that the value of i has changed 
  define integer var1 = 10
    define integer i = 0
    while i < var1 do
        run i = i + 1
    end while
    write i
EXAMPLE 3
Create a separate function from the main program and then call that function to the main program using the execute statement 
   empty function function1()
    write "Hello world of functions"
    end function1 

    start MainProgram
         run function1()
    end MainProgram
