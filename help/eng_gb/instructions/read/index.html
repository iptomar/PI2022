Read

Reads one or more variables defined in memory from the console. The instruction also allows us to define new variables for memory whose value is initialized by the console. It is also possible to set a message to be displayed when the value is requested.
Flowchart Pseudocode
Read [<Type>] <var>[,<var> …] [<message>]
• [<Type>] –Data type (INTEGER | REAL | LOGICAL | TEXT) [Optional] • <var> – Name of the defined variable or to be defined in memory. The variable name must follow the following rules:
o It cannot be a reserved word
o Must start with a letter (a-z; A-Z)
o May contain digits (0-9)
o Must be significant of what it stores
• [<message>] – message to be displayed before reading the variables [Optional]
