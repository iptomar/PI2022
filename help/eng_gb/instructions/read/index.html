Read

Reads one or more variables defined in memory from the console. The instruction also allows defining new variables for memory whose value is initialized by the console. It is also possible to set a message to be displayed when the value is requested.

Syntax 

Read[<Tipo>]<var>[,<var>...][<menssage>]


<TYPE> – Data type (integer | real | logical | text) [Optional] • – Name of the defined variable or to be defined in memory. The variable name must follow the following rules:

<var>
 o Cannot be a reserved word
 o Must start with a letter (a-z; A-Z)
 o May contain digits (0-9)
 o Must be meaningful from what it stores
[<menssage>] – message to be displayed before reading the variables [Opcional]

Examples

Read the phrase "Hello World" 
 Read "Hello World"
Read a text variable(type omission) 
 Read text txt = "Algorithmi"
 read txt
