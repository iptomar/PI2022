Iterating a set of values

Lets you iterate over a set of values within the range [ex1 , ex2[ where values are separated by ex3. The beginning limit of the iterator is included and the ending limit is not. The statement lets you define an iteration variable or use an existing numeric variable.

Syntax 

ITERATE <[TYPE]> <var> FROM <ex1> UP TO <ex2> STAGE <ex3>

<algorithm instructions> 

Fim


<TYPE> – numeric variable type ( INTEGER | REAL) - Optional.
<var> iteration variable name.
<ex1> computational expression to determine the initial value.
<ex2> computational expression to determine the final value.
<ex3> computational expression to determine the separation between the range values.
