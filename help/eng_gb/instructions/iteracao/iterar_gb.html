Iteration of a set of values

Lets you iterate over a set of values ​​within the range [ex1 , ex2[ where values ​​are separated by ex3. The starting iterator limit is included and the ending limit is not. The instruction allows you to define an iteration variable or use an existing numeric variable.

(Type) - numeric variable type ( INTEGER | REAL) - Opcional
(var) - iteration variable name
(ex1) - computational expression to determine the initial value
(ex2) - computational expression to determine the final value
(ex3) - computational expression to determine the separation between the range values

Example 1 - count in ascending order

Algorithm that prints in the console the integers between 0 and 10 in ascending order
Fluxogram pseudocode
begin MainProgram
for integer n from 0 to 10 step 11
write n + " "
end for
end MainProgram

Example 2 - count in ascending order

Algorithm that prints in the console the integers between 0 and 10 in ascending order
Fluxogram pseudocode
begin MainProgram
write "Sum of Pairs [0,10]\n"
define integer soma = 0
for integer i from 1 to 6 step 1
define integer f = 2 * i
write soma + " + " + f
execute soma = soma + f
write " = " + soma + "\n"
end for
