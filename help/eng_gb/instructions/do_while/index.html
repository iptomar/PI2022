Do... while

Allows you to repeat a set of instructions based on a condition that is evaluated at the end of the repetition.

Sintaxe 

DO <instructions> WHILE <logic expression></logic>

<TYPE> – data type (inteiro | real | logico | texto)

<logic expression> – logical expression

<instruction> – algorithm instructions

Example

Define integer variable 0 
 define integer n = 0
write and execute 
 write n + " " 
 execute n = n + 1
while a variable is less than 10 
 while n < 10
