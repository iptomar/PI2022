Conditional decision

Allows you to split the stream in two based on a condition

Syntax 

SE <logical expression>
 <instructions>
 [ ELSE ]
 <instructions>
 FIM


<logic expression> – logic expression
<instructions> – algorithm instructions
The ELSE symbol of the decision structure is optional and the decision structures can be chained together

Examples


check if a student has a Pass or Fail grade 
if grade > 10 Pass else Fail
Check if a number is positive negative or null 
 if num > 0 then positive if num == 0 then null else Negative
