Conditional decision

Allows you to split the stream in two based on a condition

Syntax 

SE <logical expression>
 <instructions>
 [ SENÃO ]
 <instruções>
 FIM


<logic expression> – logic expression
<instructions> – algorithm instructions
O simbolo SENÃO da estrutura de decisão é opcional e as estruturas de decisão podem ser encadeadas

Exemples


verificar se um aluno tem nota Aprovada ou Reprovada 
se nota > 10 aprovado senão Reprovado
Verificar se um número é positivo negativo ou nulo 
 se num > 0 então positivo se num == 0 entao nulo senão Negativo
