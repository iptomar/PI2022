Asterisk square

Problema Fluxograma Algorithmi
Build an algorithm that prints an asterisk square on the console. The height of the square should be entered by the user.

 
    begin MainProgram
        read integer height "Height : "
        for integer i from 0 to height  step 1
            for integer j from 0 to height  step 1
                write "*"
            end for
            write "\n"
        end for
    end MainProgram
