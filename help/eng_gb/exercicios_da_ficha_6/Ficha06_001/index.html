10 multiplication tables

Problema Fluxograma Algorithmi
Build an algorithm that prints the multiplication table of a number.

 
    begin MainProgram
        for integer n from 1 to 11  step 1
            write "\nTabuada do " + n + "\n"
            for integer i from 1 to 11  step 1
                write n + " x " + i + " = " + i * n + "\n"
            end for
        end for
    end MainProgram
