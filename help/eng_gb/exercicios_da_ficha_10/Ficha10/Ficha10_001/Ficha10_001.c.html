Infinite cycle - Read positive number

Write a program that prompts the user for a positive number. If the user enters a wrong value, an error message should be printed.

C Language c11

// Programador          Ant0ni0 M@ns0 - Ant0ni0 M@ns0

#include <stdio.h>
#include <string.h>
#include <math.h>
//Main Function
int main() {
    int i = 0 ;
    while(1 ){
        printf("Positive number : ");
        scanf("%d", &i);
        if(i > 0 ){
            break;
        }
        printf("Error in reading\n\n");
    }
    printf("Entered value = %d",i );
    return 0;
}
