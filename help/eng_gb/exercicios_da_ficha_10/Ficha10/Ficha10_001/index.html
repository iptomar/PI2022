Infinite cycle - Read positive number

Problema Fluxograma Algorithmi
Write a program that prompts the user for a positive number. If the user enters a wrong value, an error message should be printed.

 
    inicio ProgramaPrincipal
        definir inteiro i = 0
        enquanto verdadeiro faz
            escrever "Positive number : "
            ler i
            se i > 0 então
                quebrar
            fim se
            escrever "Error in reading\n\n"
        fim enquanto
        escrever "Entered value = " + i
    fim ProgramaPrincipal
