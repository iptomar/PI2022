Number of Digits

Problema Fluxograma Algorithmi
Build an algorithm that prints the number of digits of a positive integer entered by the user.

 
    begin MainProgram
        read integer num "Positive number\t: "
        write "Number of digits \t= " + elementsOf(num + "")
    end MainProgram
