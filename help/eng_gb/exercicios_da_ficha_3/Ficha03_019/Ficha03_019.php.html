Sine Cosine and Tangent

Construa um algoritmo que solicite ao utilizador um ângulo em radianos e de seguida imprima o seu seno, coseno e tangente.

PHP 5.3 +

<?php

//Main Function
function main() {
    echo  "Angle (rad)     : " ;
    $ang = floatval(isset($_POST["ang"])? $_POST["ang"] : isset($_GET["ang"]) ? $_GET["ang"] : 0);
    
    echo $ang . "<br/>";// print input 
    echo  "Sin         = " . ( sin( $ang )  ) . "\n" ;
    echo  "Cosine         = " . ( cos( $ang )  ) . "\n" ;
    echo  "Tangent     = " . ( tan( $ang )  ) ;
}
