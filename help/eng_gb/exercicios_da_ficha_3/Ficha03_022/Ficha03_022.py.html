Hyperbolic Trigonometry

Build an algorithm that prompts the user for an angle in radians with respect to the hyperbola and then prints out its sine, cosine, and hyperbolic tangent.

Python 3

# Programmer           Ant0ni0 M@ns0 - Ant0ni0 M@ns0

import math

ang =  float(input("Angle \t\t\t: "))
print( "Hyperbolic sine \t= ", math.sinh( ang ))
print( "Hyperbolic Cosine \t= ", math.cosh( ang ))
print( "Hyperbolic tangent \t= ", math.tanh( ang )  , end="")
