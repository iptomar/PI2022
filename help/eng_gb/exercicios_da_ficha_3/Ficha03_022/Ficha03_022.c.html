Hyperbolic Trigonometry

Build an algorithm that prompts the user for an angle in radians with respect to the hyperbola and then prints out its sine, cosine, and hyperbolic tangent.

C Language c11

// Programmer           Ant0ni0 M@ns0 - Ant0ni0 M@ns0

#include <stdio.h>
#include <string.h>
#include <math.h>
//Main Function
int main() {
    printf("Angle \t\t\t: ");
    float ang;
    scanf("%f", &ang);
    printf("Hyperbolic sine \t= %f\n",sinh( ang ) );
    printf("Hyperbolic Cosine \t= %f\n",cosh( ang ) );
    printf("Hyperbolic tangent \t= %f",tanh( ang ) );
    return 0;
}
