Hyperbolic Trigonometry

Problema Fluxograma Algorithmi
Build an algorithm that prompts the user for an angle in radians with respect to the hyperbola and then prints out its sine, cosine, and hyperbolic tangent.

 
    begin MainProgram
        read real ang "Angle \t\t\t: "
        write "Hyperbolic sine \t= " + sinh(ang) + "\n"
        write "Hyperbolic Cosine \t= " + cosh(ang) + "\n"
        write "Hyperbolic tangent \t= " + tanh(ang)
    end MainProgram
