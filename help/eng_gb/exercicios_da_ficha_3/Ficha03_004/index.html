Pythagorean Theorem - Operators

Problema Fluxograma Algorithmi
Build an algorithm that calculates the hypotenuse (c) of a right triangle given its base (a) and its height (b).

 
    begin MainProgram
        write "Pythagorean Theorem\n"
        read real a "Catet a \t: "
        read real b "Catet b \t: "
        define real c = (a * a + b * b) ^ 0.5
        write "Hypotnusa \t: " + c
    end MainProgram
