Maximum and Minimum - Numbers

Problema Fluxograma Algorithmi
Build an algorithm that prompts the user for an integer and a real number and then prints them out in an ordered fashion using the functions: min - calculation of minimum max - maximum calculation

 
    begin MainProgram
        read integer n1 "integer number\t: "
        read real n2 "real number \t: "
        write "Ordered Numbers"
        write "\n" + min(n1 , n2)
        write "\n" + max(n1 , n2)
    end MainProgram
