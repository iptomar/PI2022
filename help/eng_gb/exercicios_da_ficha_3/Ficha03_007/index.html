Pythagorean Theorem - Operators - Potency - Root

Problema Fluxograma Algorithmi
Build an algorithm that calculates the hypotenuse (c) of a right triangle given its base (a) and its height (b).

 
    begin MainProgram
        write "Pythagorean Theorem\n"
        write "Catet a \t: "
        read real a
        write "Catet b \t: "
        read real b
        define real c = sqrt(pow(a , 2) + b ^ 2)
        write "Hypotnusa \t: " + c
    end MainProgram
