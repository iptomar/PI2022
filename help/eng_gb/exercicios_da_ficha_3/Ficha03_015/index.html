Absolute Value - Reais

Problema Fluxograma Algorithmi
Build an algorithm that prompts the user for a number and then prints the absolute value of that number

 
    begin MainProgram
        read real x "X : "
        write "| " + x + " | = " + abs(x)
    end MainProgram
