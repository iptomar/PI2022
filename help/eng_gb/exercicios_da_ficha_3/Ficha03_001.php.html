Potency

Build an algorithm that calculates the value of the power whose base and exponent are entered by the user. The program should print the calculation done by: - ^ operator - pow function

PHP 5.3 +

<?php

//Main Function
function main() {
    echo  "Base         : " ;
    $base = floatval(isset($_POST["base"])? $_POST["base"] : isset($_GET["base"]) ? $_GET["base"] : 0);
    
    echo $base . "<br/>";// print input 
    echo  "Exponent      : " ;
    $expo = floatval(isset($_POST["expo"])? $_POST["expo"] : isset($_GET["expo"]) ? $_GET["expo"] : 0);
    
    echo $expo . "<br/>";// print input 
    echo  "\nPotency     = " . ( pow( $base , $expo )  ) ;
    echo  "\nOperator     = " . ( $base ^ $expo  ) ;
}

main();
