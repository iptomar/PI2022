Square Root

Problema Fluxograma Algorithmi
Build an algorithm that calculates the square root value of a number entered by the user The program should print the calculation done by: - sqrt function - pow function - operator ^

 
    begin MainProgram
        read real x "Value of X\t: "
        write "\nRoot \t\t= " + sqrt(x)
        write "\nPotency \t= " + pow(x , 0.5)
        write "\nOperator \t= " + x ^ 0.5
    end MainProgram
