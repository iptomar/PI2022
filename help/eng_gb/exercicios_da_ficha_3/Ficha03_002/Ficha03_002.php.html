Square Root

Build an algorithm that calculates the square root value of a number entered by the user The program should print the calculation done by: - sqrt function - pow function - operator ^

PHP 5.3 +

<?php

//Main Function
function main() {
    echo  "Value of X    : " ;
    $x = floatval(isset($_POST["x"])? $_POST["x"] : isset($_GET["x"]) ? $_GET["x"] : 0);
    
    echo $x . "<br/>";// print input 
    echo  "\nRoot         = " . ( sqrt( $x )  ) ;
    echo  "\nPotency     = " . ( pow( $x , 0.5 )  ) ;
    echo  "\nOperator     = " . ( $x ^ 0.5  ) ;
}
