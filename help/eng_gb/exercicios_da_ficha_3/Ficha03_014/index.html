Absolute Value - Integers

Problema Fluxograma Algorithmi
Build an algorithm that prompts the user for a number and then prints the absolute value of that number

 
    begin MainProgram
        read integer x "  X \t: "
        write "| X | \t= " + abs(x)
    end MainProgram
Input Output

-7 X : -7 | X | = 7
-9 X : -9 | X | = 9
4 X : 4 | X | = 4

10 X : 10 | X | = 10
1 X : 1 | X | = 1
-1 X : -1 | X | = 1
-6 X : -6 | X | = 6
-3 X : -3 | X | = 3
