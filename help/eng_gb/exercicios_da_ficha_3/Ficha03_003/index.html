Problema Fluxograma Algorithmi
Build an algorithm that calculates the value of the exponential of a number entered by the user. The program should print the calculation done by: - exp function - pow function - operator ^

 
    begin MainProgram
        read real x "Value of X\t: "
        write "Exponential \t= " + exp(x)
        write "\nPotency \t= " + pow(_E , x)
        write "\nOperator \t= " + _E ^ x
    end MainProgram
