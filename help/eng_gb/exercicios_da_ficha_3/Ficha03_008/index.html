Logarithm base 10

Problema Fluxograma Algorithmi
Build an algorithm that asks when using a number of then prints its base 10 logarithm.

 
    begin MainProgram
        read real x "Value of X\t: "
        define real logX = log(x)
        write "\nLogarithm 10 \t= " + logX
    end MainProgram
