Cosine Arc

Problema Fluxograma Algorithmi
Build an algorithm that asks the user for the value of a sine and then for the value of its arc.

 
    begin MainProgram
        read real x "Cosine \t: "
        write "Arch \t= " + acos(x)
    end MainProgram
