Potency <=> Logarithm

Build a program that: 1 - prompts the user for a real number 2 - calculates the base 10 logarithm of the number 3 - Calculates the base 10 power of the logarithm 4 - Make a critical comment about the algorithm's result

PHP 5.3 +

<?php

//Main Function
function main() {
    echo  "Value of X        : " ;
    $x = floatval(isset($_POST["x"])? $_POST["x"] : isset($_GET["x"]) ? $_GET["x"] : 0);
    
    echo $x . "<br/>";// print input 
    $logX = 1/log(10)*log( $x ) ;
    echo  "Log10(X)         = " . $logX  ;
    $pow10 = 10 ^ $logX ;
    echo  "\nPow10(Log10(X))     = " . $pow10  ;
}

main();
