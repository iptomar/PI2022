Potency <=> Logarithm

Build a program that: 1 - prompts the user for a real number 2 - calculates the base 10 logarithm of the number 3 - Calculates the base 10 power of the logarithm 4 - Make a critical comment about the algorithm's result

C Language c11

#include <stdio.h>
#include <string.h>
#include <math.h>
//Main Function
int main() {
    printf("Value of X\t\t: ");
    float x;
    scanf("%f", &x);
    float logX = log10( x ) ;
    printf("Log10(X) \t\t= %f",logX );
    float pow10 = pow( 10 , logX ) ;
    printf("\nPow10(Log10(X)) \t= %f",pow10 );
    return 0;
}
