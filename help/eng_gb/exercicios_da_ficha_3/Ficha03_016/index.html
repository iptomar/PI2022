Whole Part

Problema Fluxograma Algorithmi
Build an algorithm that prompts the user for a real number and then prints only the integer part

 
    begin MainProgram
        read real x "Real Number \t: "
        define integer pInt = int(x)
        write "Whole Part \t= " + pInt
    end MainProgram
