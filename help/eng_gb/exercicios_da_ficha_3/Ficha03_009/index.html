Natural logarithm - base E

Problema Fluxograma Algorithmi
Build an algorithm that asks when using a number of then prints its natural logarithm.

 
    begin MainProgram
        read real x "Value of X\t: "
        define real y = ln(x)
        write "\nln(X) \t\t= " + y
    end MainProgram
