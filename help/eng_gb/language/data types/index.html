Define Variables

Data types

In algorithmi save values in variables, these values can be of different types so it is necessary to choose the type of value, the following table shows the possible types of values that algorithmi supports.

The type <Void> is used to define a function that does not calculate values (procedure).

Type values Example
Integers An integer data type is a non-decimal number between -2,147,483,648 and 2,147,483,647. -9223372036854775808 --> 9223372036854775807
real 4.9E-324 --> 1.797693134862315E308 Real numbers with 64 bits
Logic A Boolean represents two possible states: TRUE or FALSE. True and False
Text A string of characters, such as "Hello world!".
A string can be any text enclosed in quotation marks.		 characters defined in quotes “”
void A NULL data type variable is a variable that has no value assigned to it. <void>
