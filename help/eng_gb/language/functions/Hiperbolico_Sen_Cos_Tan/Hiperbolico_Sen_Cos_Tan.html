Hyperbolic Sen, Cos and Tan

Algorithm that requests an angle in radians with respect to the hyperbola and then prints its hyperbolic sine, cosine and tagent
Flowchart pseudocode 
    start MainProgram
            read real ang "Angle \t\t\t: "
        write "Hyperbolic sine \t= " + sinh(ang) + "\n"
        write "Hyperbolic cosine \t= " + cosh(ang) + "\n"
        write "Hyperbolic tangent \t=" + tanh(ang)
            end MainProgram
