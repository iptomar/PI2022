Sine, Cosine and Tangent

Algorithm that requests an angle and prints its sine, cosine and tangent.
Flowchart Pseudocode 
    start MainProgram
                read real ang "Angle (rad) \t: "
                write "Sine \t\t= " + sin(ang) + "\n"
                write "Cosine \t\t=" + cos(ang) + "\n"
                    write "Tangent \t=" + tan(ang)
                    end MainProgram
