Algorithmi

A linguagem algorítmica GAL é uma linguagem baseada em marcadores. Todos os programas são compilados e executados utilizando a GAL. Os marcadores GAL podem ser traduzidos para qualquer linguagem natural. A linguagem foi definida tendo como principal objetivo o ensino de programação utilizando algoritmos. Os algoritmos são escritos com palavras da língua nativa dos alunos que são traduzidos e executados em GAL.
