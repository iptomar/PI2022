Problema Fluxograma Algorithmi
Build an algorithm that sums two integers entered by the user

 
    inicio ProgramaPrincipal
        ler inteiro n1 "1st number : "
        ler inteiro n2 "2nd number : "
        definir inteiro sum = n1 + n2
        escrever n1 + " + " + n2 + " = " + sum
    fim ProgramaPrincipal
