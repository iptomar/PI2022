Operators with multi instructions

Problema Fluxograma Algorithmi
Build an algorithm that prompts the user for a real number and an integer and then prints the result of the five operators: + Sum - Subtraction * Multiplication / Division ^ Potentiation

 
    inicio ProgramaPrincipal
        ler real n1 "Real number : "
        ler inteiro n2 "Integer  : "
        definir real r1 = 0,r2 = 0,r3 = 0,r4 = 0
        executar r1 = n1 + n2, r2 = n1 - n2, r3 = n1 * n2, r4 = n1 / n2
        escrever n1 + " + " + n2 + " = " + r1 , "\n" + n1 + " - " + n2 + " = " + r2 , "\n" + n1 + " * " + n2 + " = " + r3 , "\n" + n1 + " / " + n2 + " = " + r4
    fim ProgramaPrincipal
