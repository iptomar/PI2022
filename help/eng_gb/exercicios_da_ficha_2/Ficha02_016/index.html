Real Operators

Problema Fluxograma Algorithmi
Build an algorithm that prompts the user for two integers and then prints the result of the six operators: + Sum - Subtraction * Multiplication / Division ^ Potentiation

 
    inicio ProgramaPrincipal
        ler real n1 "1st number :"
        ler real n2 "2nd number :"
        escrever n1 + " + " + n2 + " = " + (n1 + n2)
        escrever "\n" + n1 + " - " + n2 + " = " + (n1 - n2)
        escrever "\n" + n1 + " * " + n2 + " = " + (n1 * n2)
        escrever "\n" + n1 + " / " + n2 + " = " + (n1 / n2)
        escrever "\n" + n1 + " ^ " + n2 + " = " + (n1 ^ n2)
    fim ProgramaPrincipal
