Integer operators

Problema Fluxograma Algorithmi
Build an algorithm that prompts the user for two integers and then prints the result of the six operators: + Sum - subtraction * Multiplication / Division Remainder of division Potentiation

 
    inicio ProgramaPrincipal
        ler inteiro n1 "1st numer :"
        ler inteiro n2 "2nd number :"
        escrever n1 + " + " + n2 + " = " + (n1 + n2)
        escrever "\n" + n1 + " - " + n2 + " = " + (n1 - n2)
        escrever "\n" + n1 + " * " + n2 + " = " + (n1 * n2)
        escrever "\n" + n1 + " / " + n2 + " = " + (n1 / n2)
        escrever "\n" + n1 + " % " + n2 + " = " + (n1 % n2)
        escrever "\n" + n1 + " ^ " + n2 + " = " + (n1 ^ n2)
    fim ProgramaPrincipal
