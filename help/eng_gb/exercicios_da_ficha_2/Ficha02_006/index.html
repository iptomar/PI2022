Writing with a comma

Build an algorithm that asks the user - his name - his age and then print out a greeting with his name and age. NOTE: In Algorithmi the comma is used to separate several arguments from the command EXAMPLE: write "hello" , " world" IS EQUIVALENT to: write "ola" write "world
Fluxograma Algorithmi 
    inicio ProgramaPrincipal
        ler texto name "What is your name \t: "
        ler inteiro age "What is your age \t: "
        escrever "Hello " + name , " of " + age + " years"
    fim ProgramaPrincipal
