Problema Fluxograma Algorithmi
Build an algorithm that prints the following figure

 
    inicio ProgramaPrincipal
        executar line(3)
        executar line(5)
        executar line(7)
        executar line(5)
        executar line(3)
    fim ProgramaPrincipal

    funcao vazio line(inteiro n)
        iterar inteiro i de 0 até n  passo 1
            escrever "*"
        fim iterar
        escrever "\n"
    fim line
