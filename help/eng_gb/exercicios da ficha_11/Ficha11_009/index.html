Triangle - line and triangle

Problema Fluxograma Algorithmi
Build a program that prints the triangle in the figure. The height must be entered by the user. Write the function: empty triangle(integer height) Use the function: empty line(integer n)

 
    inicio ProgramaPrincipal
        ler inteiro height "Height : "
        executar triangle(height)
    fim ProgramaPrincipal

    funcao vazio line(inteiro n)
        iterar inteiro i de 0 até n  passo 1
            escrever "*"
        fim iterar
        escrever "\n"
    fim line

    funcao vazio triangle(inteiro height)
        iterar inteiro i de 1 até height + 1  passo 1
            executar line(i)
        fim iterar
    fim triangle
