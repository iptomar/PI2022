Digits of a number

Problema Fluxograma Algorithmi
Build an algorithm to print the digits of an integer on the console.

 
    begin MainProgram
        read integer num "Number : "
        write "Digits = "
        while num > 0 do
            write num % 10 + " "
            execute num = num / 10
        end while
    end MainProgram
