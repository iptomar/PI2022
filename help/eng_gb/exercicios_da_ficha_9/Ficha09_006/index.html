Odd numbers in a range - optimized

Problema Fluxograma Algorithmi
Build an algorithm that asks the user for the height of a person in meters. The height of a person varies between 20 cm and 3 meters.

 
    begin MainProgram
        read integer inf "Inferior limit : "
        if inf % 2 == 0 then
            execute inf = inf + 1
        end if
        read integer sup "Upper limit : "
        while inf <= sup do
            write inf + " "
            execute inf = inf + 2
        end while
    end MainProgram
