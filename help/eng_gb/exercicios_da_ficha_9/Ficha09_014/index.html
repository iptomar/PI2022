Decimal => Binary

Problema Fluxograma Algorithmi
Write an algorithm to convert a user-entered number to binary.

 
    begin MainProgram
        read integer num "Decimal : "
        define string binary = ""
        while num > 0 do
            execute binary = num % 2 + binary
            execute num = num / 2
        end while
        write "Binary = " + binary
    end MainProgram
