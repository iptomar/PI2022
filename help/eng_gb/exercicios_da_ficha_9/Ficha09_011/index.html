Bits of a number

Problema Fluxograma Algorithmi
Build an algorithm to print the bits of an integer on the console.

 
    begin MainProgram
        read integer num "Number : "
        write "Bits = "
        while num > 0 do
            write num % 2 + " "
            execute num = num / 2
        end while
    end MainProgram
