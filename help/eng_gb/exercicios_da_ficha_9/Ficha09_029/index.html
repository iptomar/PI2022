Perfect Number 

In mathematics, a perfect number is a natural number for which the sum of all its natural proper divisors (excluding itself) is equal to the number itself.

Build an algorithm that prompts the user for a number and then checks whether it is perfect or common.

6 = 1 + 2 + 3
28 = 1 + 2 + 4 + 7 + 18

https://en.wikipedia.org/wiki/Perfect_number


Input Output
6 Number : 6 Divisors = 1 2 3 Sum = 6 Perfect Number
28 Number : 28 Divisors = 1 2 4 7 14 Sum = 28 Perfect Number
496 Number : 496 Divisors = 1 2 4 8 16 31 62 124 248 Sum = 496 Perfect Number
12 Number : 12 Divisors = 1 2 3 4 6 Sum = 16 Common Number

25 Number : 25 Divisors = 1 5 Sum = 6 Common Number
