Divisors of a number

Problema Fluxograma Algorithmi
Build an algorithm that prints on the console the divisors of a number. The number must be entered by the user.

 
    begin MainProgram
        read integer num "Numero : "
        define integer i = 1
        write "Divisors : "
        while i < num do
            if num % i == 0 then
                write i + " "
            end if
            execute i = i + 1
        end while
    end MainProgram
