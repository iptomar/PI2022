Divisors of a number

Build an algorithm that prints on the console the divisors of a number. The number must be entered by the user.

PHP 5.3 +

<?php
// Programmer           Ant0ni0 M@ns0 - Ant0ni0 M@ns0

//Main Function
function main() {
    echo  "Numero : " ;
    $num = intval(isset($_POST["num"])? $_POST["num"] : isset($_GET["num"]) ? $_GET["num"] : 0);
    
    echo $num . "<br/>";// print input 
    $i = 1 ;
    echo  "Divisors : " ;
    while($i < $num ){
        if($num % $i == 0 ){
            echo $i  .  " " ;
        }
        $i = $i + 1 ;
    }
}

main();
