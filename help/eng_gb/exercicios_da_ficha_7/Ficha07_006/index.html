Odd-Productory

Problema Fluxograma Algorithmi
Build a program that calculates the value of the formula. The value of n must be entered by the user.

 
    begin MainProgram
        read integer n "Number of terms : "
        define integer product = 1
        for integer i from 1 to n + 1  step 1
            define integer f = 2 * i - 1
            execute product = product * f
        end for
        write "Result = " + product
    end MainProgram
