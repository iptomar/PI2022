Functions by Branches

Problema Fluxograma Algorithmi
Build a program that calculates the value of the function g whose value of x is entered by the user.

 
    begin MainProgram
        read real x "x : "
        write "g(" + x + ")= "
        if x < 1 then
            execute x = -x ^ 2 + 1
        else
            execute x = x - 1
        end if
        write x
    end MainProgram
