Formula Productory

Problema Fluxograma Algorithmi
Build a program that calculates the value of the formula. The value of n must be entered by the user. NOTE: cubic root = potency of 1/3

 
    begin MainProgram
        read integer n "Number of terms : "
        define real product = 1
        for integer i from 1 to n + 1  step 1
            define real f = pow(i , 1.0 / 3)
            execute product = product * f
        end for
        write "Result = " + product
    end MainProgram
