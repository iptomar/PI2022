Problema Fluxograma Algorithmi
Implement the function f that makes any integer even by the formula f(x).

 
    begin MainProgram
        read integer x "x : "
        write "f(" + x + ")= "
        if x % 2 == 0 then
            execute x = x + 1
        end if
        write x
    end MainProgram
