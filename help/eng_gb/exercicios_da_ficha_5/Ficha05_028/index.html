Primo Number ? 

Build a program that prompts the user for a number and then tells the user whether the number is "Prime" or "Common".
A number is prime if it is divisible only by 1 and itself. The others are common.


Input Output
67 Number : 67 Cousin
7 Number : 7 Cousin
89 Number : 89 Cousin
98 Number : 98 Common
75 Number : 75 Common
44 Number : 44 Common
91 Number : 91 Common
45 Number : 45 Common
81 Number : 81 Common
71 Number : 71 Cousin
