Triangular Numbers 

A triangular number is one that is obtained by multiplying three consecutive integers.
Examples:
1st triangular = 1 x 2 x 3 = 6
2nd triangular = 2 x 3 x 4 = 24

Build an algorithm that asks the user how many numbers he wants to calculate, then calculate and print these triangular numbers on the console.
