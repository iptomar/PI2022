Minimum Grade

Build an algorithm that asks the user for the staff grade of 5 students. Then print out the minimum score.

Pseudocódigo

// Programmer           Ant0ni0 M@ns0 - Ant0ni0 M@ns0

    begin MainProgram
        define integer minimum = 999
        for integer i from 1 to 6  step 1
            read integer grade i + "º grade : "
            if grade < minimum then
                execute minimum = grade
            end if
        end for
        write "Nota mínima = " + minimum
    end MainProgram
