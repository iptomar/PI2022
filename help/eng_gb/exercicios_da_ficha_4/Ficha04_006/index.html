Positive Null Negative - see 3

Problema Fluxograma Algorithmi
Build an algorithm that checks if a number entered by the user is positive, negative, or zero.

 
    inicio ProgramaPrincipal
        ler inteiro num "Number \t: "
        se num >= 0 então
            se num > 0 então
                escrever "Positive"
            senão
                escrever "Null"
            fim se
        senão
            escrever "Negative"
        fim se
    fim ProgramaPrincipal
Input Output
0 Number : 0 Null
10 Number : 10 Positive
-8 Number : -8 Negative
-3 Number : -3 Negative
-9 Number : -9 Negative
-4 Number : -4 Negative
-10 Number : -10 Negative
-6 Number : -6 Negative
5 Number : 5 Positive
4 Number : 4 Positive
