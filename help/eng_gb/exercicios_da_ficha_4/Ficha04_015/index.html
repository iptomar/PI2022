Logical Operators - Disjunction - &&

Problema Fluxograma Algorithmi
The result of an expression containing a logical operator is ALWAYS a logical value. && disjunction also called logical multiplication 0 x 0 = 0 1 x 0 = 0 0 x 1 = 0 1 x 1 = 1

 
    inicio ProgramaPrincipal
        escrever "\n" + verdadeiro + "\t&&\t" + verdadeiro + "\t= " + (verdadeiro && verdadeiro)
        escrever "\n" + verdadeiro + "\t&&\t" + falso + "\t\t= " + (verdadeiro && falso)
        escrever "\n" + falso + "\t\t&&\t" + verdadeiro + "\t= " + (falso && verdadeiro)
        escrever "\n" + falso + "\t\t&&\t" + falso + "\t\t= " + (falso && falso)
    fim ProgramaPrincipal
Input Output
verdadeiro && verdadeiro = verdadeiro verdadeiro && falso = falso falso && verdadeiro = falso falso && falso = falso
