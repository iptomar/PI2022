Open Interval

Problema Fluxograma Algorithmi
Build an algorithm that checks whether the staff note entered by the user is valid or invalid. Use the || (OR , OR) operator. NOTE 1: For open intervals the OR operator is used. NOTE 2: When the range is open we use the operators > and <

 
    inicio ProgramaPrincipal
        ler inteiro grade "Grade \t: "
        se grade < 0 || grade > 20 então
            escrever "Invalid Grade"
        senão
            escrever "Valid Grade"
        fim se
    fim ProgramaPrincipal
Input Output
24 Grade : 24 Invalid Grade
-10 Grade : -10 Invalid Grade
3 Grade : 3 Valid Grade
8 Grade : 8 Valid Grade
-6 Grade : -6 Invalid Grade
-5 Grade : -5 Invalid Grade
12 Grade : 12 Valid Grade
4 Grade : 4 Valid Grade
9 Grade : 9 Valid Grade
7 Grade : 7 Valid Grade
