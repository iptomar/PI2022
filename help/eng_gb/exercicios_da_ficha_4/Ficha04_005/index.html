Null Positive Negative - see 2

Problema Fluxograma Algorithmi
Build an algorithm that checks if a number entered by the user is positive, negative, or zero.

 
    inicio ProgramaPrincipal
        ler inteiro num "Number \t: "
        se num == 0 então
            escrever "Null"
        senão
            se num > 0 então
                escrever "Positive"
            senão
                escrever "Negative"
            fim se
        fim se
    fim ProgramaPrincipal
Input Output
0 Number : 0 Null
-5 Number : -5 Negative
-1 Number : -1 Negative
6 Number : 6 Positive
1 Number : 1 Positive
8 Number : 8 Positive
3 Number : 3 Positive
4 Number : 4 Positive
-9 Number : -9 Negative
7 Number : 7 Positive
