Rounding a real number 

Build an algorithm that prompts the user for a real number, positive or negative, and then prints the integer that corresponds to its roundness.

NOTE: 
Rounding is always done to the nearest integer.

NOTE if the value is negative:
If the value of the decimal part is >= 0.5 it moves to the previous integer

-2.676 Real Number : -2.676 Integer Number = -3

4.065 Real Number : 4.065 Integer Number = 4
