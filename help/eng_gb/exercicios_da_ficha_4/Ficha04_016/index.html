Closed Interval

Problema Fluxograma Algorithmi
Build an algorithm that checks whether the staff note entered by the user is valid or invalid. NOTE 1: For closed intervals the operator E is used NOTE 2: When the interval is closed we use the operators >= and <=

 
    inicio ProgramaPrincipal
        ler inteiro grade "Grade \t: "
        se grade >= 0 && grade <= 20 então
            escrever "Valid Grade"
        senão
            escrever "Invalid Grade"
        fim se
    fim ProgramaPrincipal
Input Output
27 Grade : 27 Invalid Grade
-3 Grade : -3 Invalid Grade
3 Grade : 3 Valid Grade
25 Grade : 25 Invalid Grade
-5 Grade : -5 Invalid Grade
14 Grade : 14 Valid Grade
11 Grade : 11 Valid Grade
24 Grade : 24 Invalid Grade
5 Grade : 5 Valid Grade
28 Grade : 28 Invalid Grade
