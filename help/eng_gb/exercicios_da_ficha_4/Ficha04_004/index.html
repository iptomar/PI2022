Positive Null Negative - see 1

Problema Fluxograma Algorithmi
Build an algorithm that checks whether a number entered by the user is positive, negative, or zero. NOTE: to decide between three values only two comparisons are needed.

 
    inicio ProgramaPrincipal
        ler inteiro num "Number \t: "
        se num > 0 então
            escrever "Positive"
        senão
            se num == 0 então
                escrever "Null"
            senão
                escrever "Negative"
            fim se
        fim se
    fim ProgramaPrincipal
Input Output
0 Number : 0 Null
-6 Number : -6 Negative
10 Number : 10 Positive
-4 Number : -4 Negative
-2 Number : -2 Negative
-10 Number : -10 Negative
3 Number : 3 Positive
9 Number : 9 Positive
4 Number : 4 Positive
2 Number : 2 Positive
