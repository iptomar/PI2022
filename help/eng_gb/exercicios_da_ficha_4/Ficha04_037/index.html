Bisext Year

Build an algorithm that determines whether a year, entered by the user, is leap. Leap years rules: (a) All years that are multiples of 400. b) All years that are multiples of 4 and are not multiples of 100.
Fluxograma Algorithmi 
    inicio ProgramaPrincipal
        ler inteiro x "Year : "
        se x % 400 == 0 || x % 4 == 0 && x % 100 != 0 então
            escrever "Bisext"
        senão
            escrever "Common"
        fim se
    fim ProgramaPrincipal
