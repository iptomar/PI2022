Comparing real variables - WRONG

Problema Fluxograma Algorithmi
Build an algorithm that checks whether the sum of two real numbers is equal to a third.

 
    inicio ProgramaPrincipal
        escrever "Check if the sum of two real values" , "\nis equal to a third\n"
        ler real r1 "Real 1 \t: "
        ler real r2 "Real 2 \t: "
        ler real r3 "Real 3 \t: "
        escrever r1 + " + " + r2 + " == " + r3 + " => "
        se r1 + r2 == r3 então
            escrever "True"
        senão
            escrever "False"
        fim se
    fim ProgramaPrincipal
Input Output
0.1 0.2 0.3 Check if the sum of two real values is equal to a third Real 1 : 0.1 Real 2 : 0.2 Real 3 : 0.3 0.1 + 0.2 == 0.3 => False
