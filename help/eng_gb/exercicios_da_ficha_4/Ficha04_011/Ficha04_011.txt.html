Sort real numbers - Ascending

Build an algorithm that prompts the user for two real numbers and then prints them out sorted in ascending order.

Pseudocódigo

// Programador          Ant0ni0 M@ns0 - Ant0ni0 M@ns0

    inicio ProgramaPrincipal
        ler real r1 "Real 1 \t: "
        ler real r2 "Real 2 \t: "
        escrever "\nAscending order\n"
        se r1 < r2 então
            escrever r1 + "\t" + r2
        senão
            escrever r2 + "\t" + r1
        fim se
    fim ProgramaPrincipal
