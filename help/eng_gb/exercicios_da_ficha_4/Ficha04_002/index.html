Relational Operators

Problema Fluxograma Algorithmi
The result of an expression containing a relational operator is ALWAYS a logical value Relational operators: == equal != not equal > greater >= greater than or equal to < smaller <= smaller or equal

 
    inicio ProgramaPrincipal
        ler inteiro n1 "First value \t: "
        ler inteiro n2 "Second value \t\t: "
        definir logico equal = n1 == n2
        escrever "\n" + n1 + "\t==\t" + n2 + "\t=\t" + equal
        definir logico different = n1 != n2
        escrever "\n" + n1 + "\t!=\t" + n2 + "\t=\t" + different
        definir logico compare = n1 > n2
        escrever "\n" + n1 + "\t>\t" + n2 + "\t=\t" + compare
        executar compare = n1 >= n2
        escrever "\n" + n1 + "\t>=\t" + n2 + "\t=\t" + compare
        escrever "\n" + n1 + "\t<\t" + n2 + "\t=\t" + (n1 < n2)
        escrever "\n" + n1 + "\t<=\t" + n2 + "\t=\t" + (n1 <= n2)
    fim ProgramaPrincipal
