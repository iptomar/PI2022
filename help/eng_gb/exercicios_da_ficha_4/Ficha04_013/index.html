Logical Operators - Negation - !

Problema Fluxograma Algorithmi
Inverse of a logical value !true = false !false = true NOTE: Logical values can be console values: 0 - represents false 1 - is true

 
    inicio ProgramaPrincipal
        escrever "\n! " + verdadeiro + " \t= " + !verdadeiro
        escrever "\n! " + falso + " \t= " + !falso

            escrever "\nVery Good !"
