Ideal weight 

Build an algorithm that taking as input the height (in meters) and sex (m/f) of a person, calculates his ideal weight (in kilograms) , using the following Formulas:
 - for men: (72.7 * height) - 58 
 - for women: (62.1 * height) - 44.7

The program should print the integer closest to the ideal weight
