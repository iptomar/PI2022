Odd or Even game 

Build a program to play "Odd or Even".
The program should:
 - prompt the user for a number of fingers.
 - generate a random number in the interval [0,5].
The computer wins if the sum of the two numbers is an even number or the player wins otherwise.


Input Output
2 Odd or Even game Number of fingers : 2 Computer Chosen : 4 2 + 4 = 6 => PAIR The computer won
5 Odd or Even game Number of fingers : 5 Computer Chosen : 4 5 + 4 = 9 => YIELD The player won
0 Odd or Even game Number of fingers : 0 Computer Chosen : 4 0 + 4 = 4 => PAIR The computer won
4 Odd or Even game Number of fingers : 4 Computer Chosen : 4 4 + 4 = 8 => PAIR The computer won
1 Odd or Even game Number of fingers : 1 Computer Chosen : 4 1 + 4 = 5 => YIELD The player won
3 Odd or Even game Number of fingers : 3 Computer Chosen : 4 3 + 4 = 7 => YIELD The player won
