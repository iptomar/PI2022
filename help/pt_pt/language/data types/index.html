Tipo de dados

Em algorithmi guardasse valores em variaveis, esses valores podem ser de diferentes tipos logo é necessario escolher o tipo de valor, a tabela seguinte mostra os possiveis tipos de valor que o algorithmi suporta.
 O tipo <vazio> é utilizado para definir uma função que não calcula valores (procedimento).
Tipos Descrição Exemplos
inteiro Um tipo de dado inteiro é um número não decimal entre -2.147.483.648 e 2.147.483.647. -9223372036854775808 --> 9223372036854775807
real 4.9E-324 --> 1.797693134862315E308 Números reais com 64 bits
Logico Um Boolean representa dois estados possíveis: TRUE ou FALSE. verdadeiro e falso
Texto Uma sequência de personagens, como "Hello world!".
Uma sequência pode ser qualquer texto dentro daspas.		 caracteres definidos entres aspas “”
vazio Uma variável de tipo de dados NULL é uma variável que não tem valor atribuído a ele. <vazio>
