Altura de uma Pessoa

Construa um algoritmo que solicite ao utilizador a altura de uma pessoa em metros. A altura de uma pessoa varia entre 20cm e 3 metros.

Pseudocódigo

// Programmer           Ant0ni0 M@ns0 - Ant0ni0 M@ns0

    begin MainProgram
        define real altura = 0.0
        do
            write "Altura : "
            read altura
        while altura < 0.2 || altura > 3.0
        write "Altura = " + altura
    end MainProgram
