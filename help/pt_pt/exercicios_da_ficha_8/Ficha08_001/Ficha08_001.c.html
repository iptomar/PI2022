Construa um algoritmo que imprima na consola os numeros naturais com 1 digito. Utilize uma intrução de iteracao.

#include <stdio.h>
#include <string.h>
#include <math.h>
//Main Function
int main() {
    int n = 0 ;
    for( int n = 0; n < 10; n = n + 1 ) {
        printf("%d ",n );
    }
    return 0;
}
