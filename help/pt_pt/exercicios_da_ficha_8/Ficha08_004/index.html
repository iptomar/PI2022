Numero Positivo

Problema Fluxograma Algorithmi
Construa um algoritmo que solicite ao utilizador um numero inteiro maior que zero.

 
    begin MainProgram
        define integer n = 0
        do
            write "n : "
            read n
        while n <= 0
        write "Valor de n = " + n
    end MainProgram
