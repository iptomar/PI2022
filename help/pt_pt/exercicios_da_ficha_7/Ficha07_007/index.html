Produtório Fórmula

Problema Fluxograma Algorithmi
Construa um programa que calcule o valor da fórmula. O valor de n deve ser introduzido pelo utilizador. NOTA: raiz cúbica = poteência de 1/3

 
    begin MainProgram
        read integer n "Número de termos : "
        define real produto = 1
        for integer i from 1 to n + 1  step 1
            define real f = pow(i , 1.0 / 3)
            execute produto = produto * f
        end for
        write "Resultado = " + produto
    end MainProgram
