Produtório Fórmula

Construa um programa que calcule o valor da fórmula. O valor de n deve ser introduzido pelo utilizador. NOTA: raiz cúbica = poteência de 1/3

PHP 5.3 +

<?php
// Programmer           Ant0ni0 M@ns0 - Ant0ni0 M@ns0

//Main Function
function main() {
    echo  "Número de termos : " ;
    $n = intval(isset($_POST["n"])? $_POST["n"] : isset($_GET["n"]) ? $_GET["n"] : 0);
    
    echo $n . "<br/>";// print input 
    $produto = 1 ;
    for( $i = 1; $i < $n + 1; $i = $i + 1 ) {
        $f = pow( $i , 1.0 / 3 ) ;
        $produto = $produto * $f ;
    }
    echo  "Resultado = " . $produto  ;
}

main();
