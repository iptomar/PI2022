Somar múltiplos de 3 

Construa um algoritmo que solicite ao utilizador dois valores que representam os limites de um intervalo.
O algoritmo deve imprimir a soma dos múltiplos de 3 desse intervalo, incluindo os limites.

NOTA:
o primeiro valor pode ser superior ao segundo


Input Output
4 -5 Somar múltiplos de 3 Limite : 4 Limite : -5 Soma 3 [-5 , 4] = 0
4 -2 Somar múltiplos de 3 Limite : 4 Limite : -2 Soma 3 [-2 , 4] = 3
-2 -3 Somar múltiplos de 3 Limite : -2 Limite : -3 Soma 3 [-3 , -2] = -3
-4 -1 Somar múltiplos de 3 Limite : -4 Limite : -1 Soma 3 [-4 , -1] = -3
5 -4 Somar múltiplos de 3 Limite : 5 Limite : -4 Soma 3 [-4 , 5] = 0
-2 -4 Somar múltiplos de 3 Limite : -2 Limite : -4 Soma 3 [-4 , -2] = -3
-4 -5 Somar múltiplos de 3 Limite : -4 Limite : -5 Soma 3 [-5 , -4] = 0
0 -1 Somar múltiplos de 3 Limite : 0 Limite : -1 Soma 3 [-1 , 0] = 0
-3 0 Somar múltiplos de 3 Limite : -3 Limite : 0 Soma 3 [-3 , 0] = -3
3 0 Somar múltiplos de 3 Limite : 3 Limite : 0 Soma 3 [0 , 3] = 3
