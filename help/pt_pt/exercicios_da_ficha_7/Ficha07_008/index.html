Problema Fluxograma Algorithmi
Contrua um programa que calcule o valor da função g cuja valor de x é introduzido pelo utilizador.

 
    begin MainProgram
        read real x "x : "
        write "g(" + x + ")= "
        if x < 1 then
            execute x = -x ^ 2 + 1
        else
            execute x = x - 1
        end if
        write x
    end MainProgram
