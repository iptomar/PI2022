Somatório 5 pares

Problema Fluxograma Algorithmi
Soma o primeiros 5 pares NOTA: par(i) = 2*i

 
    begin MainProgram
        write "Somatório Pares [0,10]\n"
        define integer soma = 0
        for integer i from 1 to 6  step 1
            define integer f = 2 * i
            write soma + " + " + f
            execute soma = soma + f
            write " = " + soma + "\n"
        end for
        write "\nSoma = " + soma
    end MainProgram
Input Output
Somatório Pares [0,10] 0 + 2 = 2 2 + 4 = 6 6 + 6 = 12 12 + 8 = 20 20 + 10 = 30 Soma = 30
