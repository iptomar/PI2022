Torna par

Problema Fluxograma Algorithmi
Implemente a função f que torna qualquer número inteiro par pela fórmula f(x).

 
    begin MainProgram
        read integer x "x : "
        write "f(" + x + ")= "
        if x % 2 == 0 then
            execute x = x + 1
        end if
        write x
    end MainProgram
