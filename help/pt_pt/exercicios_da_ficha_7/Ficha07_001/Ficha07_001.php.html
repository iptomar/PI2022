Somatório

Soma os primeiros 5 números inteiros positivos. NOTA: Elemento neutro da soma = zero

PHP 5.3 +

<?php
// Programmer           Ant0ni0 M@ns0 - Ant0ni0 M@ns0

//Main Function
function main() {
    echo  "Somatório\n" ;
    $soma = 0 ;
    for( $i = 1; $i < 6; $i = $i + 1 ) {
        echo $soma  .  " + " . $i  ;
        $soma = $soma + $i ;
        echo  "      = " . $soma  .  "\n" ;
    }
    echo  "\nTotal = " . $soma  ;
}

main();
