Somatório

Problema Fluxograma Algorithmi
Soma os primeiros 5 números inteiros positivos. NOTA: Elemento neutro da soma = zero

 
    begin MainProgram
        write "Somatório\n"
        define integer soma = 0
        for integer i from 1 to 6  step 1
            write soma + " + " + i
            execute soma = soma + i
            write " \t = " + soma + "\n"
        end for
        write "\nTotal = " + soma
    end MainProgram
Input Output
Somatório 0 + 1 = 1 1 + 2 = 3 3 + 3 = 6 6 + 4 = 10 10 + 5 = 15 Total = 15
