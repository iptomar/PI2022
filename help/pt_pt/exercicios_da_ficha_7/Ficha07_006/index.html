Problema Fluxograma Algorithmi
Construa um programa que calcule o valor da fórmula. O valor de n deve ser introduzido pelo utilizador.

 
    begin MainProgram
        read integer n "Número de termos : "
        define integer produto = 1
        for integer i from 1 to n + 1  step 1
            define integer f = 2 * i - 1
            execute produto = produto * f
        end for
        write "Resultado = " + produto
    end MainProgram
