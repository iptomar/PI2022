Divisores de um numero

Problema Fluxograma Algorithmi
Construa um algoritmo que imprima na consola os divisores de um numero. O numero deve ser introduzido pelo utilizador.

 
    begin MainProgram
        read integer num "Numero : "
        define integer i = 1
        write "Divisores : "
        while i < num do
            if num % i == 0 then
                write i + " "
            end if
            execute i = i + 1
        end while
    end MainProgram
