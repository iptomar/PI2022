Dígitos de um numero

Problema Fluxograma Algorithmi
Contrua um algoritmo para imprimir na consola os dígitos de um numero inteiro.

 
    begin MainProgram
        read integer num "Numero : "
        write "Digitos = "
        while num > 0 do
            write num % 10 + " "
            execute num = num / 10
        end while
    end MainProgram
