Decimal => Binario

Problema Fluxograma Algorithmi
Escreva um algoritmo para converter para binario um numero introduzido pelo utilizador.

 
    begin MainProgram
        read integer num "Decimal : "
        define string binario = ""
        while num > 0 do
            execute binario = num % 2 + binario
            execute num = num / 2
        end while
        write "Binario = " + binario
    end MainProgram
