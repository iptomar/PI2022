Numeros Ímpares num intervalo - optimizado

Construa um algoritmo que solicite ao utilizador a altura de uma pessoa em metros. A altura de uma pessoa varia entre 20cm e 3 metros.

JavaScript

// Programmer           Ant0ni0 M@ns0 - Ant0ni0 M@ns0

//Main Function
function main() {
    document.write("Limite inferior : " );// print text
    inf = parseInt(prompt("Limite inferior : " ,"0") );
    
    document.write(inf + "\n");// print input 
    if(inf % 2 == 0 ){
        inf++;
    }
    document.write("Limite superior : " );// print text
    sup = parseInt(prompt("Limite superior : " ,"0") );
    
    document.write(sup + "\n");// print input 
    while(inf <= sup ){
        document.write(inf + " " );
        inf +=2;
    }
}
