Numeros Ímpares num intervalo - optimizado

Construa um algoritmo que solicite ao utilizador a altura de uma pessoa em metros. A altura de uma pessoa varia entre 20cm e 3 metros.

C Language c11

// Programmer           Ant0ni0 M@ns0 - Ant0ni0 M@ns0

#include <stdio.h>
#include <string.h>
#include <math.h>
//Main Function
int main() {
    printf("Limite inferior : ");
    int inf;
    scanf("%d", &inf);
    if(inf % 2 == 0 ){
        inf++;
    }
    printf("Limite superior : ");
    int sup;
    scanf("%d", &sup);
    while(inf <= sup ){
        printf("%d ",inf );
        inf +=2;
    }
    return 0;
}
