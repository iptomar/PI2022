Numeros Ímpares num intervalo - Força bruta

Problema Fluxograma Algorithmi
Construa um algoritmo que imprima na consola os números ímpares de um itervalo introduzido pelo utilizador.

 
    begin MainProgram
        read integer inf "Limite inferior : "
        read integer sup "Limite superior : "
        while inf <= sup do
            if inf % 2 != 0 then
                write inf + " "
            end if
            execute inf = inf + 1
        end while
    end MainProgram
