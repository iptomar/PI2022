Bits de um número

Problema Fluxograma Algorithmi
Contrua um algoritmo para imprimir na consola os bits de um numero inteiro.

 
    begin MainProgram
        read integer num "Numero : "
        write "Bits = "
        while num > 0 do
            write num % 2 + " "
            execute num = num / 2
        end while
    end MainProgram
