Ciclo infinito - Ler número positivo

Escreva um programa que solicite ao utilizador um número positivo. Se o utilizador introduzir um valor errado deve ser impressa uma mensagem de erro.

PHP 5.3 +

<?php
// Programador          Ant0ni0 M@ns0 - Ant0ni0 M@ns0

//Main Function
function main() {
    $i = 0 ;
    while(true ){
        echo  "Número positvo : " ;
        $i = intval(isset($_POST["i"])? $_POST["i"] : isset($_GET["i"]) ? $_GET["i"] : 0);
        
        echo $i . "<br/>";// print input 
        if($i > 0 ){
            break;
        }
        echo  "Erro na leitura\n\n" ;
    }
    echo  "Valor introduzido = " . $i  ;
}

main();
