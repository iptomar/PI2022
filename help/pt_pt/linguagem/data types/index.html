Tipo de dados

A linguagem GAL é tipada e todas as variáveis em memória possuem um tipo. O tipo VOID é utilizado para definir uma função que não calcula valores (procedimento).

GAL Descrição Valores pt-PT
INTEGER Números inteiros com 64 bits -9223372036854775808 9223372036854775807 inteiro
REAL Números reais com 64 bits 1.797693134862315E308 4.9E-324 real
LOGIC Valores lógicos verdadeiro e falso Logico
TEXT Conjunto de caracteres da tabela unicode caracteres definidos entres aspas “” Texto
