Triângulo - Linha

Problema Fluxograma Algorithmi
Construa um algoritmo que imprima o triângulo da figura. A altura deve ser introduzida pelo utilizador. Utilize a funcao: vazio linha(inteiro n)

 
    inicio ProgramaPrincipal
        ler inteiro altura "Altura : "
        iterar inteiro i de 1 até altura + 1  passo 1
            executar linha(i)
        fim iterar
    fim ProgramaPrincipal

    funcao vazio linha(inteiro n)
        iterar inteiro i de 0 até n  passo 1
            escrever "*"
        fim iterar
        escrever "\n"
    fim linha
