Nota Mínima

Problema Fluxograma Algorithmi
Contrua um algoritmo que solicite ao utilizador a nota de pauta de 5 alunos. De seguida imprima a nota mínima.

 
    begin MainProgram
        define integer minimo = 999
        for integer i from 1 to 6  step 1
            read integer nota i + "º nota : "
            if nota < minimo then
                execute minimo = nota
            end if
        end for
        write "Nota mínima = " + minimo
    end MainProgram
