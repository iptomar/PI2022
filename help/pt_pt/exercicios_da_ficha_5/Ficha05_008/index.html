Pares num intervalo - Decidir

Problema Fluxograma Algorithmi
Escreva um algoritmo que solicite ao utilizador os limites de um intervalo. De seguida imprima todos os números pares desse intervalo incluindo os limites.

 
    begin MainProgram
        read integer imin "Minimo : "
        read integer imax "Maximo : "
        for integer i from imin to imax + 1  step 1
            if i % 2 == 0 then
                write i + " "
            end if
        end for
    end MainProgram
