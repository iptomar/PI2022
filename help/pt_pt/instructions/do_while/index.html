Faz... enquanto

Permite repetir um conjunto de instruções baseadas numa condição que é avaliada no final da repetição.

Sintaxe 

FAZ <instruçoes> ENQUANTO <expressao lógica></lógica>

<expressao lógica> – expressão lógica

<instruçoes> – instruções do algoritmo

Exemplos

Define variavel inteira 0 
 definir inteiro n = 0
ecreve e executa 
 escrever n + " " 
 executar n = n + 1
Enquanto variavel for menor que 10 
 enquanto n < 10
