Decisão condicional

Permite dividir o fluxo em dois baseado numa condição

Sintaxe 

SE <expressao lógica>
 <instruçoes>
 [ SENÃO ]
 <instruçoes>
 FIM


<expressao lógica> – expressão lógica
<instruçoes> – instruções do algoritmo

O simbolo SENAO da estrutura de decisão é opcional e as estruturas de decisão podem ser encadeadas

Exemplos


verificar se um aluno tem nota Aprovada ou Reprovada 
se nota > 10 aprovado senão Reprovado
Verificar se um número é positivo negativo ou nulo 
 se num > 0 então positivo se num == 0 entao nulo senão Negativo
