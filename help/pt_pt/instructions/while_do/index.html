Repetição com controlo inicial – Enquanto…faz

Permite repetir um conjunto de instruções baseadas numa condição que é avaliada no início da repetição.

Sintaxe 

ENQUANTO <expressao lógica>

<instruçoes do algoritmo> 

Fim

Exemplos

Define variavel inteira 0 
 definir inteiro n = 0
Enquanto variavel for menor que 10 
 enquanto n < 10
ecreve e executa 
 escrever n + " " 
 executar n = n + 1
