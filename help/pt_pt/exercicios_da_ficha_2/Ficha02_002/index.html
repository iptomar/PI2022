Ler com mensagem

Construa um algoritmo que solicite ao utilizador o seu nome de se seguida imprima uma saudação com o seu nome. A leitura do nome deve ser precedida de uma mensagem na consola.
Fluxograma Algorithmi 
    begin MainProgram
        write "Qual o seu nome : "
        read string nome
        write "Olá " + nome
    end MainProgram
