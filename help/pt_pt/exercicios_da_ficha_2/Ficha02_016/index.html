Operadores Reais

Construa um algoritmos que solicite ao utilizador dois números inteiros e de seguida imprima o resultado dos seis operadores: + Soma - Subtração * multiplicação / Divisão ^ Potenciação
Fluxograma Algorithmi 
    begin MainProgram
        read real n1 "1º número :"
        read real n2 "2º número :"
        write n1 + " + " + n2 + " = " + (n1 + n2)
        write "\n" + n1 + " - " + n2 + " = " + (n1 - n2)
        write "\n" + n1 + " * " + n2 + " = " + (n1 * n2)
        write "\n" + n1 + " / " + n2 + " = " + (n1 / n2)
        write "\n" + n1 + " ^ " + n2 + " = " + (n1 ^ n2)
    end MainProgram
