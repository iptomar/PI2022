Operadores Reais

Problema Fluxograma Algorithmi
Construa um algoritmos que solicite ao utilizador dois números inteiros e de seguida imprima o resultado dos seis operadores: + Soma - Subtração * multiplicação / Divisão ^ Potenciação

 
    inicio ProgramaPrincipal
        ler real n1 "1º número :"
        ler real n2 "2º número :"
        escrever n1 + " + " + n2 + " = " + (n1 + n2)
        escrever "\n" + n1 + " - " + n2 + " = " + (n1 - n2)
        escrever "\n" + n1 + " * " + n2 + " = " + (n1 * n2)
        escrever "\n" + n1 + " / " + n2 + " = " + (n1 / n2)
        escrever "\n" + n1 + " ^ " + n2 + " = " + (n1 ^ n2)
    fim ProgramaPrincipal
