Operadores com multi instruções

Construa um algoritmo que solicite ao utilizador um número real e um inteiro e de seguida imprima o resultado dos cinco operadores: + Soma - Subtração * multiplicação / Divisão ^ Potenciação
Fluxograma Algorithmi 
    begin MainProgram
        read real n1 "Número real    : "
        read integer n2 "Número inteiro : "
        define real r1 = 0,r2 = 0,r3 = 0,r4 = 0
        execute r1 = n1 + n2, r2 = n1 - n2, r3 = n1 * n2, r4 = n1 / n2
        write n1 + " + " + n2 + " = " + r1 , "\n" + n1 + " - " + n2 + " = " + r2 , "\n" + n1 + " * " + n2 + " = " + r3 , "\n" + n1 + " / " + n2 + " = " + r4
    end MainProgram
