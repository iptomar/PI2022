Operadores com multi instruções

Construa um algoritmo que solicite ao utilizador um número real e um inteiro e de seguida imprima o resultado dos cinco operadores: + Soma - Subtração * multiplicação / Divisão ^ Potenciação

C Language c11

#include <stdio.h>
#include <string.h>
#include <math.h>
//Main Function
int main() {
    printf("Número real    : ");
    float n1;
    scanf("%f", &n1);
    printf("Número inteiro : ");
    int n2;
    scanf("%d", &n2);
    float r1 = 0 ;
    float r2 = 0 ;
    float r3 = 0 ;
    float r4 = 0 ;
    r1 = n1 + n2 ;
    r2 = n1 - n2 ;
    r3 = n1 * n2 ;
    r4 = n1 / n2 ;
    printf("%f + %d = %f",n1 ,n2 ,r1 );
    printf("\n%f - %d = %f",n1 ,n2 ,r2 );
    printf("\n%f * %d = %f",n1 ,n2 ,r3 );
    printf("\n%f / %d = %f",n1 ,n2 ,r4 );
    return 0;
}
