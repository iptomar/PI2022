Operadores Inteiros e reais

Problema Fluxograma Algorithmi
Construa um algoritmo que solicite ao utilizador um número real e um inteiro e de seguida imprima o resultado dos cinco operadores: + Soma - Subtração * multiplicação / Divisão ^ Potenciação

 
    inicio ProgramaPrincipal
        ler real n1 "1º número :"
        ler inteiro n2 "2º número :"
        escrever n1 + " + " + n2 + " = " + (n1 + n2)
        escrever "\n" + n1 + " - " + n2 + " = " + (n1 - n2)
        escrever "\n" + n1 + " * " + n2 + " = " + (n1 * n2)
        escrever "\n" + n1 + " / " + n2 + " = " + (n1 / n2)
        escrever "\n" + n1 + " ^ " + n2 + " = " + (n1 ^ n2)
    fim ProgramaPrincipal
