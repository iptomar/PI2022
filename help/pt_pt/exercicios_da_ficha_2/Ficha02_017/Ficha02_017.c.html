Operadores Inteiros e reais

Construa um algoritmo que solicite ao utilizador um número real e um inteiro e de seguida imprima o resultado dos cinco operadores: + Soma - Subtração * multiplicação / Divisão ^ Potenciação

C Language c11

#include <stdio.h>
#include <string.h>
#include <math.h>
//Main Function
int main() {
    printf("1º número :");
    float n1;
    scanf("%f", &n1);
    printf("2º número :");
    int n2;
    scanf("%d", &n2);
    printf("%f + %d = %f",n1 ,n2 ,( n1 + n2 ) );
    printf("\n%f - %d = %f",n1 ,n2 ,( n1 - n2 ) );
    printf("\n%f * %d = %f",n1 ,n2 ,( n1 * n2 ) );
    printf("\n%f / %d = %f",n1 ,n2 ,( n1 / n2 ) );
    printf("\n%f ^ %d = %f",n1 ,n2 ,( pow( n1 , n2 ) ) );
    return 0;
}
