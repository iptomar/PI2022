Operadores inteiros

Construa um algoritmo que solicite ao utilizador dois números inteiros e de seguida imprima o resultado dos seis operadores: + Soma - Subtração * multiplicação / Divisão % Resto da divisão ^ Potenciação

Pseudocódigo

// Programmer           Ant0ni0 M@ns0 - Ant0ni0 M@ns0

    begin MainProgram
        read integer n1 "1º número :"
        read integer n2 "2º número :"
        write n1 + " + " + n2 + " = " + (n1 + n2)
        write "\n" + n1 + " - " + n2 + " = " + (n1 - n2)
        write "\n" + n1 + " * " + n2 + " = " + (n1 * n2)
        write "\n" + n1 + " / " + n2 + " = " + (n1 / n2)
        write "\n" + n1 + " % " + n2 + " = " + (n1 % n2)
        write "\n" + n1 + " ^ " + n2 + " = " + (n1 ^ n2)
    end MainProgram
