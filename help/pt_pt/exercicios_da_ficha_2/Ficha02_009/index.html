Somar dois números multi-instrução

Construa um algoritmos que some dois números inteiros introduzidos pelo utilizador
Fluxograma Algorithmi 
    begin MainProgram
        read integer n1,n2 "Introduza dois números\n"
        write n1 + " + " + n2 + " = " , n1 + n2
    end MainProgram
