Problema Fluxograma Algorithmi
Construa um algoritmos que some dois números inteiros introduzidos pelo utilizador

 
    inicio ProgramaPrincipal
        ler inteiro n1 "1º número : "
        ler inteiro n2 "2º número : "
        executar n1 = n1 + n2
        escrever "Soma = " + n1
    fim ProgramaPrincipal
