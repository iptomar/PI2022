Construa um algoritmos que some dois números inteiros introduzidos pelo utilizador
Fluxograma Algorithmi 
    begin MainProgram
        read integer n1 "1º número : "
        read integer n2 "2º número : "
        execute n1 = n1 + n2
        write "Soma = " + n1
    end MainProgram
