Somar dois números Inteiros

Construa um algoritmos que some dois números inteiros introduzidos pelo utilizador
Fluxograma Algorithmi 
    begin MainProgram
        write "1º número :"
        read integer n1
        write "2º número :"
        read integer n2
        write "Soma = " + (n1 + n2)
    end MainProgram
