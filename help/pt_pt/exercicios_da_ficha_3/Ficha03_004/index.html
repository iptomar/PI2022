Teorema de Pitágoras - Operadores

Problema Fluxograma Algorithmi
Construa um algoritmo que calcule a hipotenusa (c) de um triangulo retângulo dada a sua base (a) e a sua altura(b).

 
    begin MainProgram
        write "Teorema de Pitágoras\n"
        read real a "Cateto a \t: "
        read real b "Cateto b \t: "
        define real c = (a * a + b * b) ^ 0.5
        write "Hipotnusa \t: " + c
    end MainProgram
