Raiz Quadrada

Construa um algoritmo que calcule o valor a raiz quadrada de um número introduzido pelo utilizador O programa deve imprimir o cálculo feito por: - função sqrt - função pow - operador ^

#include <string.h>
#include <math.h>
//Main Function
int main() {
    printf("Valor de X\t: ");
    float x;
    scanf("%f", &x);
    printf("\nRaiz \t\t= %f",sqrt( x ) );
    printf("\nPotência \t= %f",pow( x , 0.5 ) );
    printf("\nOperador \t= %f",pow( x , 0.5 ) );

}
