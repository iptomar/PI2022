Teorema de Pitágoras - Raiz Quadrada

Construa um algoritmo que calcule a hipotenusa (c) de um triangulo retângulo dada a sua base (a) e a sua altura(b).

C Language c11

#include <stdio.h>
#include <string.h>
#include <math.h>
//Main Function
int main() {
    printf("Teorema de Pitágoras\n");
    printf("Cateto a \t: ");
    float a;
    scanf("%f", &a);
    printf("Cateto b \t: ");
    float b;
    scanf("%f", &b);
    float c = sqrt( a * a + b * b ) ;
    printf("Hipotnusa \t: %f",c );
    return 0;
}
