Número de Dígitos

Problema Fluxograma Algorithmi
Construa um algoritmo que imprima o número de dígitos de um número inteiro positivo introduzido pelo utilizador.

 
    begin MainProgram
        read integer num "Número positivo\t: "
        write "Número de dígitos \t= " + elementsOf(num + "")
    end MainProgram
