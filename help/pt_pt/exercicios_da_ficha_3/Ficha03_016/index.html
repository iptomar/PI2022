Parte inteira

Problema Fluxograma Algorithmi
Construa um algoritmo que solicite ao utilizador um número real e de seguida imprima apenas a parte inteira

 
    begin MainProgram
        read real x "Número real \t: "
        define integer pInt = int(x)
        write "Parte inteira \t= " + pInt
    end MainProgram
