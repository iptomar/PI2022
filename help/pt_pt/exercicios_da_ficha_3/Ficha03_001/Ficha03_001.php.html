Potência

Construa um algoritmo que calcule o valor da potência cuja base e expoente são introduzido pelo utilizador. O programa deve imprimir o cálculo feito por: - operador ^ - função pow

PHP 5.3 +

<?php
// Programmer           Ant0ni0 M@ns0 - Ant0ni0 M@ns0

//Main Function
function main() {
    echo  "Base         : " ;
    $base = floatval(isset($_POST["base"])? $_POST["base"] : isset($_GET["base"]) ? $_GET["base"] : 0);
    
    echo $base . "<br/>";// print input 
    echo  "Expoente     : " ;
    $expo = floatval(isset($_POST["expo"])? $_POST["expo"] : isset($_GET["expo"]) ? $_GET["expo"] : 0);
    
    echo $expo . "<br/>";// print input 
    echo  "\nPotência     = " . ( pow( $base , $expo )  ) ;
    echo  "\nOperador     = " . ( $base ^ $expo  ) ;
}

main();
