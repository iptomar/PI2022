Potência

Problema Fluxograma Algorithmi
Construa um algoritmo que calcule o valor da potência cuja base e expoente são introduzido pelo utilizador. O programa deve imprimir o cálculo feito por: - operador ^ - função pow

 
    begin MainProgram
        read real base "Base \t\t: "
        read real expo "Expoente \t: "
        write "\nPotência \t= " + pow(base , expo)
        write "\nOperador \t= " + base ^ expo
    end MainProgram
