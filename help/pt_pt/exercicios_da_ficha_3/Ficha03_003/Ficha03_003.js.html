Expoencial

Construa um algoritmo que calcule o valor da exponencial de um número introduzido pelo utilizador. O programa deve imprimir o cálculo feito por: - função exp - função pow. - operador ^

JavaScript

// Programador          Ant0ni0 M@ns0 - Ant0ni0 M@ns0

//Main Function
function main() {
    document.write("Valor de X\t: " );// print text
    x = parseFloat(prompt("Valor de X\t: " ,"0.0") );
    
    document.write(x + "\n");// print input 
    document.write("Exponencial \t= " + Math.exp( x ) );
    document.write("\nPotencia \t= " + Math.pow( Math.E , x ) );
    document.write("\nOperador \t= " + Math.pow( Math.E , x ) );
}
