Expoencial

Problema Fluxograma Algorithmi
Construa um algoritmo que calcule o valor da exponencial de um número introduzido pelo utilizador. O programa deve imprimir o cálculo feito por: - função exp - função pow. - operador ^

 
    begin MainProgram
        read real x "Valor de X\t: "
        write "Exponencial \t= " + exp(x)
        write "\nPotencia \t= " + pow(_E , x)
        write "\nOperador \t= " + _E ^ x
    end MainProgram
