Valor Absoluto - Reais

Problema Fluxograma Algorithmi
Construa um algoritmo que solicite ao utilizador um número e de seguida imprima o valor absoluto desse número

 
    begin MainProgram
        read real x "X : "
        write "| " + x + " | = " + abs(x)
    end MainProgram
