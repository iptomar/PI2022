Problema Fluxograma Algorithmi
Construa um algoritmo que solicite ao utilizador o nome de duas pessoas e de seguida os imprima de forma ordenadautilizando as funções: min - cálculo do mínimo max - cálculo do máximo

 
    begin MainProgram
        read string nome1 "Nome : "
        read string nome2 "Nome : "
        write "Nomes Ordenados"
        write "\n" + min(nome1 , nome2)
        write "\n" + max(nome1 , nome2)
    end MainProgram
