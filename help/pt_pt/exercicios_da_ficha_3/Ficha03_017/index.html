Número de Caracteres

Problema Fluxograma Algorithmi
Construa um algoritmo que imprima o número de caracteres de um nome introduzido pelo utilizador.

 
    begin MainProgram
        read string nome "Nome : "
        write "Numero de caracteres = " + elementsOf(nome)
    end MainProgram
