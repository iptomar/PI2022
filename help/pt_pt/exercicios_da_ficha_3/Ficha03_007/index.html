Teorema de Pitágoras - Operadores - Potencia - Raiz

Problema Fluxograma Algorithmi
Construa um algoritmo que calcule a hipotenusa (c) de um triangulo retângulo dada a sua base (a) e a sua altura(b).

 
    begin MainProgram
        write "Teorema de Pitágoras\n"
        write "Cateto a \t: "
        read real a
        write "Cateto b \t: "
        read real b
        define real c = sqrt(pow(a , 2) + b ^ 2)
        write "Hipotnusa \t: " + c
    end MainProgram
