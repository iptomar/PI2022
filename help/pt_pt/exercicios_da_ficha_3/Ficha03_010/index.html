Potência <=> Logaritmo

Problema Fluxograma Algorithmi
Contrua uma programa que: 1 - solicite ao utilizador um número real 2 - calcule o logaritmo base 10 do número 3 - Calcule a potência base 10 do logaritmo 4 - Faça um comentário crítico acerca do resultado do algoritmo

 
    begin MainProgram
        read real x "Valor de X\t\t: "
        define real logX = log(x)
        write "Log10(X) \t\t= " + logX
        define real pow10 = 10 ^ logX
        write "\nPow10(Log10(X)) \t= " + pow10
    end MainProgram
