Jogo do par ou ímpar 

Construa um programa para jogar "Par ou ímpar".
O programa deve:
 - solicitar ao utilizador um o número de dedos.
 - gerar um número aleatório no intervalo [0,5]
O computador ganha se a soma dos dois números for um número par ou ganha o jogador no caso contrário.


Input Output
1 Jogo do par ou ímpar Número de dedos : 1 Computador escolheu : 4 1 + 4 = 5 => ÍMPAR O jogador ganhou
5 Jogo do par ou ímpar Número de dedos : 5 Computador escolheu : 4 5 + 4 = 9 => ÍMPAR O jogador ganhou
4 Jogo do par ou ímpar Número de dedos : 4 Computador escolheu : 4 4 + 4 = 8 => PAR O computador ganhou
0 Jogo do par ou ímpar Número de dedos : 0 Computador escolheu : 4 0 + 4 = 4 => PAR O computador ganhou
2 Jogo do par ou ímpar Número de dedos : 2 Computador escolheu : 4 2 + 4 = 6 => PAR O computador ganhou
3 Jogo do par ou ímpar Número de dedos : 3 Computador escolheu : 4 3 + 4 = 7 => ÍMPAR O jogador ganhou
