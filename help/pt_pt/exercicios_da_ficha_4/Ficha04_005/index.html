Nulo Positivo Negativo - ver 2

Problema Fluxograma Algorithmi
Construa um algoritmo que verifique se um numero introduzido pelo utilizador e positivo, negativo ou nulo.

 
    begin MainProgram
        read integer num "Número \t: "
        if num == 0 then
            write "Nulo"
        else
            if num > 0 then
                write "Positivo"
            else
                write "Negativo"
            end if
        end if
    end MainProgram
Input Output

-2 Número : -2 Negativo
-4 Número : -4 Negativo
10 Número : 10 Positivo
-9 Número : -9 Negativo
