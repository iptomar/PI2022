Positivo Nulo Negativo - ver 1

Problema Fluxograma Algorithmi
Construa um algoritmo que verifique se um numero introduzido pelo utilizador e positivo, negativo ou nulo. NOTA: para decidir entre três valores são necessárias apenas duas comparações.

 
    begin MainProgram
        read integer num "Número \t: "
        if num > 0 then
            write "Positivo"
        else
            if num == 0 then
                write "Nulo"
            else
                write "Negativo"
            end if
        end if
    end MainProgram
Input Output
0 Número : 0 Nulo
-5 Número : -5 Negativo
-3 Número : -3 Negativo
9 Número : 9 Positivo
3 Número : 3 Positivo
2 Número : 2 Positivo
5 Número : 5 Positivo
-6 Número : -6 Negativo
1 Número : 1 Positivo
-9 Número : -9 Negativo
