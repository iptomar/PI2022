Aprovado ou Reprovado

Construa um algoritmo que verifique se um aluno está aprovado ou reprovado dada a sua nota final. Um aluno está aprovado se a sua nota for maior ou igual a 10 valores.

C Language c11

#include <stdio.h>
#include <string.h>
#include <math.h>
//Main Function
int main() {
    printf("Nota\t: ");
    int nota;
    scanf("%d", &nota);
    if(nota >= 10 ){
        printf("Aprovado");
    }else{
        printf("Reprovado");
    }
    return 0;
}
