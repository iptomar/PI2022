Arredondar um número real 

Construa um algoritmo que solicite ao utilizador um número real, positivo ou negativo, e de seguida imprima o número inteiro que corresponde ao seu arredondamento.

NOTA: 
O Arredondamento é sempre feito para o inteiro máis próximo.

NOTA se o valor for negativo:
Se o valor da parte decimal for >= 0.5 passa para o inteiro anterior


Input Output
-8.7 Nº Real : -8.7 Nº Inteiro = -9
-5.986 Nº Real : -5.986 Nº Inteiro = -6
9.836 Nº Real : 9.836 Nº Inteiro = 10
-2.892 Nº Real : -2.892 Nº Inteiro = -3
-2.170 Nº Real : -2.17 Nº Inteiro = -2
6.490 Nº Real : 6.49 Nº Inteiro = 6
-1.432 Nº Real : -1.432 Nº Inteiro = -1
-0.303 Nº Real : -0.303 Nº Inteiro = 0
9.161 Nº Real : 9.161 Nº Inteiro = 9
1.170 Nº Real : 1.17 Nº Inteiro = 1
