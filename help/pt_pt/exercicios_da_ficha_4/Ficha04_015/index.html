Operadores Lógicos - Disjunção - &&

Problema Fluxograma Algorithmi
O resultado de uma expressão que contenha um operador Lógicos é SEMPRE um valor lógico. && disjunção também chamada de multiplicação lógica 0 x 0 = 0 1 x 0 = 0 0 x 1 = 0 1 x 1 = 1

 
    begin MainProgram
        write "\n" + true + "\t&&\t" + true + "\t= " + (true && true)
        write "\n" + true + "\t&&\t" + false + "\t\t= " + (true && false)
        write "\n" + false + "\t\t&&\t" + true + "\t= " + (false && true)
        write "\n" + false + "\t\t&&\t" + false + "\t\t= " + (false && false)
    end MainProgram
Input Output
true && true = true true && false = false false && true = false false && false = false
