ERRO 0.1 + 0.2 = 0.3

Problema Fluxograma Algorithmi
ERRO DE COMPUTAÇÃO 0.1 + 0.2 = 0.3 NOTA 1: As operações sobre reais podem ser arredondadas. NOTA 2: As comparações entre reais NUNCA devem ser feitas com o operador de igualdade ( == )

 
    begin MainProgram
        define real r1 = 0.1
        define real r2 = 0.2
        if r1 + r2 == 0.3 then
            write "Soma Correta"
        else
            write "Soma Errada "
            write r1 + r2
        end if
    end MainProgram

Soma Errada 0.30000000000000004
