Par ou ímpar 

Construa um algoritmo que verifique se um valor introduzido pelo utilizador é "Par" ou "Ímpar"

NOTA:
um número é par se for divisível por 2.


Input Output

-19 Número : -19 Ímpar
49 Número : 49 Ímpar
43 Número : 43 Ímpar
3 Número : 3 Ímpar
61 Número : 61 Ímpar
98 Número : 98 Par
-11 Número : -11 Ímpar
96 Número : 96 Par
13 Número : 13 Ímpar
