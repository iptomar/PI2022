Intervalo Aberto

Problema Fluxograma Algorithmi
Construa um algoritmo que verifique se a nota de pauta introduzida pelo utilizador é válida ou inválida. Utilize o operador || (OU , OR). NOTA 1: Para intervalos abertos utiliza-se o operador OU. NOTA 2: Quando o intervalo é aberto utilizam-se os operadores > e <

 
    begin MainProgram
        read integer nota "Nota \t: "
        if nota < 0 || nota > 20 then
            write "Nota inválida"
        else
            write "Nota válida"
        end if
    end MainProgram
