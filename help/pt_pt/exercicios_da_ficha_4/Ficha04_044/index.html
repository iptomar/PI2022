Raízes de uma equação de 2ª grau - II 

Construa um programa que calcule o valor das raízes de uma equação de 2º grau.

Uma equação tem:

0 Raizes - se o delta for menor que ZERO

1 Raiz	 - se o delta for igual a ZERO 
          a raiz é dada pela fórmula -b / 2a  

2 Raizes - caso contrário
          as raízes são dadas pela fórmula de Bhaskara 

NOTA:
Utilize uma precisão de 5 CASAS DECIMAIS
