Intervalo Fechado

Problema Fluxograma Algorithmi
Construa um algoritmo que verifique se a nota de pauta introduzida pelo utilizador é válida ou inválida. NOTA 1: Para intervalos fechados utiliza-se o operador E NOTA 2: Quando o intervalo é fechado utilizam-se os operadores >= e <=

 
    begin MainProgram
        read integer nota "Nota \t: "
        if nota >= 0 && nota <= 20 then
            write "Nota válida"
        else
            write "Nota inválida"
        end if
    end MainProgram
