Números Reais

Problema Fluxograma Algorithmi
Os números reais são definidos pela norma IEEE 754 - são representados por 64 bites - São representados na forma SINAL EXPOENTE FRAÇÃO NOTA: Os números reais são arrendondados.

 
    begin MainProgram
        define real r1 = 0.1
        define real r2 = 0.2
        define real r3 = 0.1 + 0.2
        write "0.1 + 0.2 = " + r3
        define real r4 = 0.3 - 0.2
        write "\n0.3 - 0.2 = " + r4
        define real r5 = 0.01234567890123456789
        write "\n0.01234567890123456789 = " + r5
    end MainProgram
Input Output
0.1 + 0.2 = 0.30000000000000004 0.3 - 0.2 = 0.09999999999999998 0.01234567890123456789 = 0.012345678901234568
