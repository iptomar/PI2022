Índice de Massa Corporal

O IMC (índice de massa corporal) é dado pela razão entre o peso (kilogramas) e o quadrado altura ao quadrado, de uma pessoa. Contrua um programa que solicite ao utilizador o se peso e a sua altura e de seguida imprima a sua classificação: IMC Classificação < 18.5 Magro [18.5 , 25.0[ Normal [25.0 , 30.0[ Gordo [30.0 - 35.0[ Obeso > 35.0 Extremamente Obeso
Fluxograma Algorithmi 
    begin MainProgram
        read real peso "Peso (kg) \t: "
        read real alt "Altura (m) \t: "
        define real imc = peso / (alt ^ 2)
        write "IMC \t\t= " + imc
        write "\nClassificação \t= "
        if imc < 18.5 then
            write "Magro"
        else
            if imc < 25 then
                write "Normal"
            else
                if imc < 30 then
                    write "Gordo"
                else
                    if imc < 35 then
                        write "Obeso"
                    else
                        write "Extremamente obeso"
                    end if
                end if
            end if
        end if
    end MainProgram
