Ordenar números reais - Crescente

Problema Fluxograma Algorithmi
Construa um algoritmo que solicte ao utilizador dois números reais aos utilizador e de seguida os imprima ordenados por ordem crescente.

 
    begin MainProgram
        read real r1 "Real 1 \t: "
        read real r2 "Real 2 \t: "
        write "\nOrdem crescente\n"
        if r1 < r2 then
            write r1 + "\t" + r2
        else
            write r2 + "\t" + r1
        end if
    end MainProgram
